
  # Digital News Website Design

  This is a code bundle for Digital News Website Design. The original project is available at https://www.figma.com/design/bEwBkrmRg4pMLhcAiNYwvM/Digital-News-Website-Design.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

  ## Deploying to GitHub Pages

  1. First, make sure you update the `<USERNAME>` and `<REPOSITORY_NAME>` inside your `package.json` file.
  2. Install all dependencies, including the `gh-pages` module, by running `npm i`.
  3. Run the following command to deploy directly to GitHub:
     ```bash
     npm run deploy
     ```
  This command creates a production build inside the `dist` folder and then automatically pushes it to the `gh-pages` branch of your repository.
